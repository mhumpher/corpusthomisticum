Corpus Thomisticum Search Engine (v0.1)

Welcome to the Corpus Thomisticum Search Engine. This site is meant to make it easier to search the Corpus Thomisticum Opera Omnia of St. Thomas Aquinas.

Enter the search terms in the box below. The default setting for the search engine is currently to make use of regular expressions. If you are unfamiliar with regular expressions, you can find some information here. 

At present only search a single term or use an or search as "term1|term2|term3".

 

This site is not associated with the Corpus Thomisticum Project. Its sole purpose is to make using that site easier and more accessible.
